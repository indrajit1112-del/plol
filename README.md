# Price Bot

A chatbot webapp that gives component prices with fuzzy search.

## Features
- Fuzzy search for CPUs, GPUs, RAM, SSDs
- Exclude keywords with `-` (e.g. `DDR5 -rgb`)
- Indian rupee formatting

## Run locally
```bash
streamlit run streamlit_app.py
```

## Deploy
Connect this repo to [Streamlit Cloud](https://share.streamlit.io) and set the main file to `streamlit_app.py`.
